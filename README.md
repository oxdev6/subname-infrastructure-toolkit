# ENS Subname Infrastructure Toolkit

Grant-level toolkit to launch ENS subname ecosystems with smart contracts, APIs, SDK, and management dashboard.

## Monorepo layout

- `contracts/` - Hardhat contracts for permissioned and signature-based subname issuance
- `backend/` - NestJS API scaffold
- `dashboard/` - Next.js app scaffold
- `sdk/` - TypeScript SDK scaffold
- `docs/` - architecture and roadmap docs

## Quick start

```bash
cd contracts
npm install
npm test
```
